1. This project validates the GUI elements of Google Search homepage then searches for a keyword and validates that search results are displayed.
2. The location where this project is saved becomes the value for the constant USERDIRECTORY in the java file.
3. I have used the chrome driver version 96.
4. The results will be stored in USERDIRECTORY/Results/<Current Timestamp>/Google_Search_Test_<Passed/Failed>.docx
5. All the external jars are in external jar folder, please add before running.
